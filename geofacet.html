How to Create a Panel Chart from Geo Data

using the R package *geofacet*

Jun 3, 2020 • Nick McCarty • 1 min read

data visualization   panel chart   geo data   R

Note: Ryan Hafen, the creator of geofacet, wrote about the package (as it compares to other tools used to visualize geo data) more thoroughly than I will here; his piece is definitely worth taking the time to read.

Panel Chart of Unites States 

#collapse-hide

######### Dependencies ##########

# install.packages("geofacet")

# https://blog.revolutionanalytics.com/2015/09/resizing-plots-in-the-r-kernel-for-jupyter-notebooks.html
library(repr)
options(repr.plot.width = 14)

library(ggplot2)
library(geofacet)

######### Plot ##########

ggplot(state_ranks, aes(variable, rank, fill = variable)) +
  geom_col() +
  coord_flip() +
  facet_geo(~ state) +
  theme_bw()

Another Panel Chart of Unites States 

#collapse-hide

######### Plot ##########

ggplot(state_unemp, aes(year, rate)) +
  geom_line() +
  facet_geo(~ state, grid = "us_state_grid2") +
  scale_x_continuous(labels = function(x) paste0("'", substr(x, 3, 4))) +
  ylab("Unemployment Rate (%)")

Panel Chart of European Union 

#collapse-hide

######### Plot ##########

ggplot(eu_gdp, aes(year, gdp_pc)) +
  geom_line(color = "steelblue") +
  facet_geo(~ name, grid = "eu_grid1", scales = "free_y") +
  scale_x_continuous(labels = function(x) paste0("'", substr(x, 3, 4))) +
  ylab("GDP Per Capita in Relation to EU Index (100)") +
  theme_bw()
